# Week 1 Practice / Reference Code

## Purpose
This practice page gives you short working examples for Week 1 topics.

Use these files to:
- see simple Python code working
- practice running Python files
- observe common syntax mistakes
- build confidence before Lab 1

This is **practice/reference code only**. It is not a graded assignment.

---

## Files in This Practice Set

### `hello_python.py`
Shows simple print statements, arithmetic, and string output.

### `debugging_examples.py`
Shows common syntax mistakes. Uncomment one example at a time to see what happens.

### `help_and_types.py`
Shows simple output and how Python stores different kinds of data.

---

## Suggested Order

1. Run `hello_python.py`
2. Run `help_and_types.py`
3. Open `debugging_examples.py`
4. Uncomment one broken example at a time and run it to see the error

---

## What to Pay Attention To

As you run the code, notice:
- how `print()` works
- how Python handles numbers and text
- how small syntax mistakes create errors
- how error messages help you debug

---

## Try This

After you run each file, try one small change such as:
- change a number in a math example
- change the text being printed
- add another print statement
- uncomment one broken example and read the error carefully
